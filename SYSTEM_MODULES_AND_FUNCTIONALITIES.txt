================================================================================
M.C DENTAL CLINIC - SYSTEM MODULES & FUNCTIONALITIES
================================================================================
Last Updated: November 2024

================================================================================
1. AUTHENTICATION & USER MANAGEMENT MODULE
================================================================================

A. Authentication System:
   - User login/logout functionality
   - User registration (patient registration)
   - Password recovery and reset
   - Email verification
   - Account claiming functionality
   - JWT token-based authentication
   - Session management

B. User Roles:
   - Admin: Full system access
   - Staff: Appointment and patient management
   - Patient: Booking and personal profile access

C. User Management:
   - User profile management (with profile images)
   - Staff member management (CRUD operations, job titles)
   - Patient management (CRUD operations, demographics)

================================================================================
2. APPOINTMENT MANAGEMENT SYSTEM MODULE
================================================================================

A. Appointment Booking:
   - Online appointment booking (for both guests and logged-in patients)
   - Calendar-based date selection
   - Time slot availability checking
   - Health screening questionnaire (required)
   - Multiple services per appointment
   - Promotion integration

B. Appointment Workflow:
   - Appointment statuses: pending, confirmed, completed, cancelled, 
     cancellation_requested, reschedule_requested, no_show
   - Cancellation requests (with reason tracking)
   - Reschedule requests (with new date/time specification)
   - Admin/Staff approval/rejection of requests
   - Appointment status tracking and updates

C. Payment Management:
   - Payment status tracking (unpaid, partial, paid)
   - Payment amount tracking
   - Payment method selection (cash, card)
   - Card type selection

13. DATABASE STRUCTURE (Supabase/PostgreSQL)
================================================================================

Core Tables:
- users: User accounts and authentication
- patients: Patient records and demographics
- doctors: Doctor profiles and information
- services: Service catalog
- schedules: Doctor-specific schedules
- appointments: Appointment records
- appointment_services: Many-to-many appointment services relationship
- medical_history: Medical records and visit history
- patient_images: Patient image gallery
- promotions: Promotional offers
- notifications: User notifications
- audit_logs: System audit trail
- gallery_cases: Before/after gallery cases
- site_settings: Site configuration
